Using Nugs Downloader

From Source

made for ash

Assuming nugs dl is already installed
  1. Open up terminal
  2. Type cd Documents
  3. Type cd NugsDL
  4. Type ./main LINK_HERE
If you need to re-install it
  1. Open terminal and type "go"
    2. If it shows a help message, skip to #5
  2. Head to go.dev
  3. Under "get started", click "macOS"
  4. Open and install the .pkg file.
  5. Go is installed! Now, install nugs:
  6. download nugs dl source code
  7. Once unzipped, open terminal and type cd Downloads
  8. cd Nugs-Downloader-main
  9. Type go build main.go structs.go
  10. once done, move the new "main" file into Documents/NugsDL
  11. You're done!
